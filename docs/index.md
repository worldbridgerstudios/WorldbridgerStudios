### HoloCell

| Constant | Error |
|----------|-------|
| Proton-electron mass ratio | 1.21 × 10⁻⁷% |
| Rydberg constant | 1.02 × 10⁻⁵% |
| Fine structure constant | 1.35 × 10⁻⁵% |
| Muon-electron mass ratio | 1.40 × 10⁻⁵% |
| Weinberg angle | 4.67 × 10⁻⁴% |
